
## What is Network?
A computer network or simply a network, is a collection of computers and other hardware components interconnected by communication channels that allow sharing of resources and information.

### Network = Node + Link
- Radio or Wireless
- Wire
- Fiber Optic

### Node
- Host (Example : PC, IP Camera, Net Printer, …)
- PCs
- Client
### Resources
- Data : File & Folder
- Hardware : Printer
- Services : Internet Sharing

## Composition of network
1. Service
2. Server
3. Data
4. Link
5. Signal
6. Transceiver
7. Protocol
