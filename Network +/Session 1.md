## What is Network?
#Network
A computer network or simply a network, is a collection of computers and other hardware components interconnected by communication channels that allow sharing of resources and information.

### Network = Node + Link
- Radio or Wireless
- Wire
- Fiber Optic

### Node
- Host (Example : PC, IP Camera, Net Printer, …)
- PCs
- Client
### Resources
- Data : File & Folder
- Hardware : Printer
- Services : Internet Sharing

## Composition of network
1. Service
2. Server
3. Data
4. Link
5. Signal
6. Transceiver
7. Protocol

## **The difference Between ==LAN & WAN==**
#difference_Lan_Wan
1. *Ownership*
2. *Cost*
3. *Speed*
4. *Security*
5. *Temporary*

|         | **Ownership**  | **Speed** | **Geographic<br>Spread** |        **Status**        |
| :-----: | :------------: | :-------: | :----------------------: | :----------------------: |
| **LAN** |    *Private*     |   *High*    |         *Limited*          |        *Permanent*         |
| **WAN** |     *Public*     |    *Low*    |        *Unlimited*         |         *Undemand*         |
| **MAN** | *Private/Public* |    *Low*    |         *Limited*          | *Permanent or<br>Undemand* |
| **PAN** |    *Private*     |    *Low*    |         *Limited*          |        *Permanent*         |

## *What is the Term "***Intranet**", "**Extranet**" & "**Internet**"

- **Internet** ***:** The internet is a public network accessible to anyone*.

- **Extranet :** *Extranets are private networks that allow external parties to access certain parts of an organization's intranet.*

- **Intranet :**  *intranets are private networks accessible only to authorized users within an organization.*
